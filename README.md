# e-plantShopping

## Paradise Nursery Shopping Application

e-plantShopping is a React and Redux Toolkit application developed as part of the Coursera Front-End Development project.

### Features

* Landing page with company information
* Product listing page with categorized houseplants
* Shopping cart functionality using Redux Toolkit
* Add products to cart
* Increase and decrease product quantity
* Remove products from cart
* Dynamic cart item counter
* Checkout button with "Coming Soon" message
* Continue shopping functionality

### Technologies Used

* React
* Redux Toolkit
* JavaScript
* CSS
* GitHub Pages

### Project Overview

Paradise Nursery is an online plant shopping application that allows users to browse different categories of houseplants and manage their shopping cart efficiently.
